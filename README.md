# Descargador de Videos de YouTube

Esta aplicación de Streamlit permite a los usuarios descargar videos de YouTube especificando la URL del video.

## Características

- Descarga videos de YouTube a través de la URL.
- Interfaz de usuario amigable construida con Streamlit.
- Maneja errores de video no encontrado o restricciones de acceso.

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado Python 3.6 o superior y pip en tu sistema.

## Instalación

Para instalar las dependencias de este proyecto, navega al directorio del proyecto y ejecuta:

```
pip install -r requirements.txt
```

Esto instalará Streamlit y yt-dlp, que son necesarios para ejecutar la aplicación.

Uso
Para iniciar la aplicación, usa el siguiente comando:

```
streamlit run app.py
```

Esto abrirá la aplicación en tu navegador predeterminado.
